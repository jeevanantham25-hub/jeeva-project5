# 📸 Photo Gallery Web Page

A responsive **Photo Gallery Web Page** created using **HTML and CSS** that displays multiple images in a grid layout with captions and modern styling.

This project is designed for beginners to understand how to create a simple gallery website using basic web technologies.

---

## 🚀 Features

* Responsive grid layout
* 20+ images included
* Image captions
* Hover animation effects
* Modern card design
* Mobile-friendly layout
* Clean and simple UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 📂 Project Structure

```
photo-gallery/

```
